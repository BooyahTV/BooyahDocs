# Liked Playbacks - Public

Request Method: GET

* */users/`userID`/liked-playblacks* = Get liked playbacks of the user
  * Response Example:

     ```js
     {
    "cursor": 0,
    "playback_list": []
    }
     ```

* */users/`userID`/liked-playblacks/count* = Get liked playbacks count of the user
  * Response Example:

     ```js
     {
    "count": 0
    }
     ```
