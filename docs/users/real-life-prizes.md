# Real Life Prices - Public

Request Method: GET

* */users/`userID`/real-life-prizes* = Get the channels followed by this user
  * Response Example:

    ```js
    {
    "prize_list": []
    }
    ```

Request Method: POST

* */users/`userID`/real-life-prizes* = Get the channels followed by this user
  * Response Example:

     ```js
     {"count":1337} // Follower Count
     ```
